###3.创建和修改规则
<pre>
<code>
用   途：创建和修改规则
地   址：/rule
method：post
参   数：
		rid - 规则id（创建时值为：0）
		uid - yoyow账号
		title - 标题 长度：20
		content - 规则内容 200
		amount - 支付金额 1000以内
		asset_id - 用于支付的资产id
		expiration_date - 过期时间
返   回：
{
	"code": "0",//成功：0，失败：5000
	"message": "操作成功",//code为5000 返回错误原因
	"data": null

</code>
